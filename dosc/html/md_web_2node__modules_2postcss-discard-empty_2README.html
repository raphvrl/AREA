<a href="https://github.com/postcss/postcss" >postcss</a>-discard-empty

‍Discard empty rules and values with PostCSS.

Install

With npm do:

npm install postcss-discard-empty --save

Example

For more examples see the tests.

Input

@font-face;
h1 {}
{color:blue}
h2 {color:}
h3 {color:red}

Output

h3 {color:red}
