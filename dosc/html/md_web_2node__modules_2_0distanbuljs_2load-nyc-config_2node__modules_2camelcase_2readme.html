‍Convert a dash/dot/underscore/space separated string to camelCase or PascalCase: foo-bar → fooBar

Install

$ npm install camelcase

Usage

const camelCase = require('camelcase');
camelCase('foo-bar');
//=> 'fooBar'
camelCase('foo_bar');
//=> 'fooBar'
camelCase('Foo-Bar');
//=> 'fooBar'
camelCase('Foo-Bar', {pascalCase: true});
//=> 'FooBar'
camelCase('--foo.bar', {pascalCase: false});
//=> 'fooBar'
camelCase('foo bar');
//=> 'fooBar'
console.log(process.argv[3]);
//=> '--foo-bar'
camelCase(process.argv[3]);
//=> 'fooBar'
camelCase(['foo', 'bar']);
//=> 'fooBar'
camelCase(['__foo__', '--bar'], {pascalCase: true});
//=> 'FooBar'

API

camelCase(input, [options])

input

Type: string string[]

String to convert to camel case.

options

Type: Object

pascalCase

Type: boolean
Default: false

Uppercase the first character: foo-bar → FooBar
